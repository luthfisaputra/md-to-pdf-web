# Markdown to PDF Web Application

A modern web application that converts Markdown documents into beautifully formatted PDF files. Built with Spring Boot, this application provides a seamless experience for creating, editing, and exporting markdown documents.

## 🎯 Overview

This project is a full-stack web application that allows users to:
- Create and manage markdown documents
- Preview markdown content in real-time
- Export documents as professional PDF files
- Store documents persistently in a database

## ✨ Features

### Document Management
- ✅ **Create Documents** - Start new markdown documents with templates
- ✅ **Edit Documents** - Full-featured markdown editor with live preview
- ✅ **Rename Documents** - Easily change document titles
- ✅ **Delete Documents** - Remove documents you no longer need
- ✅ **List Documents** - View all your documents in a sidebar

### Export Capabilities
- 📄 **Markdown to HTML** - Convert markdown to semantic HTML
- 📑 **HTML to PDF** - Generate professional PDF files with custom styling
- 🎨 **Styled Output** - PDF documents with:
  - Professional typography
  - Code syntax highlighting
  - Proper spacing and formatting
  - Custom fonts

### Auto-Save
- 💾 **Real-time Saving** - Changes are automatically saved as you type
- 🔄 **Instant Persistence** - Data stored immediately in database
